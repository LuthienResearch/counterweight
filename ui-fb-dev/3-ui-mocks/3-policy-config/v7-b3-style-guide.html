How it works ▼
1. StyleChecker is a SimpleLLMPolicy with your style guide as the judge prompt.
2. On each response, Luthien sends the output plus your style guide to a judge model.
3. If the score is below your threshold, Luthien retries with explicit constraints from the failing criteria.
4. You can adjust the threshold and retry count below.
