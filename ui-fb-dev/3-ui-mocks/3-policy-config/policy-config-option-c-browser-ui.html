What kinds of policies can I create?

Block dangerous commands

Write rules in plain English. An LLM judge evaluates them.

class SafetyPolicy(SimpleJudgePolicy): RULES = [ "Block rm -rf commands", "Block git push --force", "Block DROP TABLE statements", ]

Enforce package standards

Block pip, suggest uv. Claude Code retries automatically.

class PipBlockPolicy(SimpleJudgePolicy): RULES = [ "Block pip install. Suggest uv add.", "Block python -m pip install.", "Allow all other tool calls.", ]

Clean AI output

Transform responses with Python. No LLM judge needed.

class DeSlop(SimplePolicy): def simple_on_response_content( self, content, context ): return content.replace( "\u2014", "-" ).replace("\u2013", "-")

Domain-specific compliance

Use the LLM judge with your own rules.

class CompliancePolicy(SimpleJudgePolicy): RULES = [ "Responses must cite actual " "policy documents, not " "hallucinate guidance.", "Block responses with PII.", ]
