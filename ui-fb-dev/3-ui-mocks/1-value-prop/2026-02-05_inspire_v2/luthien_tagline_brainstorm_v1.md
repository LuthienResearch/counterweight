# Luthien Tagline Brainstorm - Round 1

## What Luthien Does (Core Value)
- Proxy between AI coding agents (Claude Code, Codex) and LLM backends
- Visibility: See every request and response in real-time
- Control: Enforce rules/policies on AI behavior
- Drop-in: Just 2 env vars to get started
- Hot-swap: Change policies without restart

## Target Audience
- Senior developers using AI 40+ hrs/week
- Teams deploying AI coding agents at scale
- Organizations needing AI governance/compliance

## Key Differentiators
- Purpose-built for AI coding agents (not general LLM apps)
- Drop-in proxy (no code changes)
- Real-time visibility + policy enforcement combined
- Open source, self-hostable
