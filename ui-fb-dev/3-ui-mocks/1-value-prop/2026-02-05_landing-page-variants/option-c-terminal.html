The problem

What your coding agent is actually doing vs. what you asked for.

agents.md → actual behavior
# Your instructions "Only modify files in src/auth/" Agent refactors 3 unrelated files, reorganizes imports, adds a feature not in spec
expected workflow → reality
# Run prompt, go get coffee Come back to scoped, reviewable changes Come back to unreviewable diff with scope creep everywhere
claude-code → codex
