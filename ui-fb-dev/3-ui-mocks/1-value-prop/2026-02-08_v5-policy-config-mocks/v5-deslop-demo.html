Live Demo: DeSlop Policy

AI writes with annoying patterns: em dashes everywhere, curly quotes, fancy punctuation nobody asked for. Luthien catches and fixes this automatically as AI writes code or text.

What gets cleaned

The DeSlop policy runs on every AI response as it streams through Luthien. It replaces AI-isms with clean text:

— → - em dash
– → - en dash
“ ” → " " curly quotes
‘ ’ → ' ' curly apostrophes
