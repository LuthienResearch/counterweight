Luthien

Policy control for AI agents

Option A: Terminal Only

Everything happens in the terminal where you already work. Luthien runs invisibly — you only see it when it blocks something.

Why this works

  • Zero context switching
  • Info appears where you already are
  • Nothing to configure — just works
  • Invisible until it matters

What you lose

  • No visual overview of all rules
  • Config lives in YAML file
  • Hard to share blocked-action history
  • No at-a-glance status
