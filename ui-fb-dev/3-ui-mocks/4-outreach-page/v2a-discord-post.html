Open-source proxy that sits between AI agents and model APIs. Intercepts requests, runs policy checks, logs everything. We're early-stage and things are rough around the edges -- your feedback genuinely makes it better.

Stuck? Jai is on Discord as a speaker/judge all weekend. Scott is lurking there too. DM us, tag @luthien, or book a 15-min call.

Project ideas:

  • Proxy-level monitor for BashArena -- The hackathon asks for "control protocols with traditional security measures, like giving the trusted model access to a firewall." Write a Luthien policy that flags suspicious commands before they run. Can you beat Ctrl-Z's 58%-to-7% result?
    Track 2 -- $1K LinuxArena prize eligible
  • Red team our policies -- The hackathon asks for "a dataset of sneaky transcripts of AIs doing very bad things." Try to get harmful actions past Luthien's built-in defenses. Document what works and what doesn't.
    Track 3
  • Interrogation protocol -- The hackathon asks to "test interrogation protocols in which the trusted model asks the untrusted model questions before executing the action." Extend Luthien's LLM-as-judge to do exactly that.
    Track 2
  • Trusted editing via proxy rewrite -- Intercept suspicious code suggestions, have a trusted model rewrite them before they reach the developer. Does the editor keep functionality while removing sabotage? Build it AND red-team it.
