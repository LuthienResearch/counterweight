Luthien: open-source proxy for AI control

Sits between AI agents and model APIs. Intercepts requests, runs policy checks, logs everything. GitHub

Built by Jai Dhyani (speaker/judge at this hackathon, talk Friday noon) and Scott Wofford. We're on Discord all weekend.

Project ideas

  • Monitor for BashArena/LinuxArena - Write a policy that flags suspicious commands before they execute. Can a proxy-level monitor beat Ctrl-Z's 58%->7% result? Track 2, LinuxArena prize eligible
  • Red team our policies - Try to get harmful actions past Luthien's defenses. Build a dataset of what works and what doesn't. Track 3
  • Interrogation protocol - Extend our LLM-as-judge to ask the untrusted model clarifying questions before allowing actions. Measure safety vs usefulness. Track 2
  • Trusted editing at the proxy layer - Intercept suspicious code, have a trusted model rewrite it, measure whether functionality survives. Track 2

More ideas + details: project-ideas.md
