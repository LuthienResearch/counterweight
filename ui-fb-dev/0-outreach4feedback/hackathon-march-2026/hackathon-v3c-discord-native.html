Luthien Proxy

Open-source proxy that sits between AI agents and model APIs. Intercepts requests, runs policy checks (regex, DLP, LLM-as-judge), logs everything. The hackathon calls for "traditional security measures like giving the trusted model access to a firewall." We built that firewall.

github.com/LuthienResearch/luthien-proxy
