# progress/AGENTS.md

This directory is the historical record of Railway work done **before** counterweight_ engaged. Read-only.

## Rules

- **Don't edit files here in place.** If something is factually wrong, add a dated correction to `/CHANGELOG.md` instead.
- **Don't add new entries here** for your own work — your work goes in `asks/` deliverables, PRs against `luthien-proxy`, or new docs at the repo root.
- **Linking is fine** — this index is exactly the kind of thing external docs should point into.

Why the asymmetry: "what was done before the vendor arrived" is a snapshot that gets stale quickly if anyone edits it. Freezing it keeps the narrative honest.
